# 브루트포스
## 1. 정의
- 가능한 모든 경우의 수를 빠짐없이 다 대입해보며 조건에 맞는 정답을 찾아내는 알고리즘 및 공격 기법을 의미한다.
- 한국어로는 무차별 대입 공격이라고 부른다.
## 2. 종류
- 크게 2가지가 있다.
  - 1) 순수 브루트포스
    - 가능한 모든 문자열을 전부 시도하는 방식이다.
    - 예를 들어 알파벳 소문자 4자리 비밀번호라면 다음과 같이 모든 경우를 하나씩 넣어본다.
      
      예 :
      aaaa
      aaab
      aaac
      ...
      zzzz
      
    - 가장 원시적인 방식이지만, 가능한 범위가 작다면 확실하게 정답을 찾을 수 있다.
    - 하지만 비밀번호의 길이가 길어지거나 사용할 수 있는 문자의 종류가 많아지면 경우의 수가 급격히 증가한다. 따라서 범위가 커질수록 시간이 매우 오래 걸린다.
  - 2) 사전 공격
    - 사람들이 자주 사용하는 비밀번호 목록을 미리 준비해두고, 그 목록을 하나씩 시도하는 방식이다.
    - 영어로는 Dictionary Attack이라고 부른다.
    - 예를 들어 다음과 같은 흔한 비밀번호를 사용한다.
      
      예 :
      password
      123456
      qwer1234
      admin
      iloveyou
      abc123
      
    - 순수 브루트포스처럼 모든 문자열을 전부 시도하지 않고, 사람들이 실제로 많이 쓰는 단어나 패턴을 우선적으로 시도한다.
    - 따라서 순수 브루트포스보다 훨씬 효율적인 경우가 많다.
# 빈도 분석
## 1. 정의
- 빈도 분석은 암호문 내에 등장하는 특정 문자나 문자열의 출현 횟수를 통계적으로 분석하여 이를 바탕으로 원래의 평문을 유추해내는 기법이다. 주로 시저 암호(카이사르)와 같이 하나의 글자가 다른 하나의 글자로 일대일 대응되는 단일 치환 암호를 무력화하는 데 사용된다.
## 2. 예시
- 어떤 범죄 조직의 통신망에서 다음과 같은 암호문을 가로챘다고 가정한다.

  예 :
  WTAA XH WXSTC XC IWT RPKT

- 이 암호문을 풀기 위해 브루트포스를 쓴다면 알파벳 26가지의 치환 경우의 수를 모두 대입해야 하지만, 빈도 분석을 적용하면 다음과 같이 풀 수 있다.
- 1단계 빈도 세기
  - 위 문장에서 T가 4번, W가 3번, X가 2번, C가 2번 등장한다.
- 2단계 가장 흔한 글자 대입
  - 세 글자로 된 단어 IWT와 WTAA가 보인다. 영어에서 가장 흔한 세 글자 단어는 THE이다.
  - 만약 IWT가 THE라면, I=T, W=H, T=E가 된다.
- 3단계 가설 검증
  - 가설대로 대입해 보면 암호문은 다음과 같이 바뀐다.
  
  예 :
  HEAA XH HXSTC XC THE RPKE

- 4단계 문맥 유추
  - HEAA는 HALL 또는 HELL일 가능성이 높다. 만약 A=L이라면 HELL이 된다. 
  - 또한 두 글자 단어 XH와 XC에서 H가 들어간 두 글자 단어는 IS, IN, IT 등이 있다.
- 결과
  - 이런 식으로 빈도와 문맥을 조합해 퍼즐을 맞추면 이 암호문이 시저 암호(15칸 이동)로 작성된 "JEWEL IS HIDDEN IN THE CAVE"라는 것을 알아낼 수 있다.
# 키 공간이 작다는 의미
## 1. 키 공간이란?
- 키 공간은 가능한 비밀번호, 암호 키, 정답 후보의 전체 개수를 의미한다.
- 영어로는 Key Space라고 부른다.
- 쉽게 말하면 시도해야 할 후보가 몇 개나 있는가를 말한다.
## 2. 키 공간이 작다는 뜻
- 키 공간이 작다는 것은 가능한 후보의 개수가 적다는 뜻이다.
- 후보가 적으면 브루트포스로 전부 시도해도 시간이 많이 걸리지 않는다.
- 그래서 공격자가 정답을 찾기 쉽다.
## 3. 예시
- 3자리 숫자 비밀번호는 다음과 같다.

  000 ~ 999

- 총 1,000개뿐이다.
