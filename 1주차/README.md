# 암호학이 필요한 이유
암호학은 디지털 시대에 정보 유출과 변조를 막는 핵심 기술이다. 암호학의 4대 목적에는  기밀성, 무결성, 인증, 부인방지가 있다. 통신 내용 암호화, 개인정보 보호, 금융 거래 보안 및 시스템 신뢰성을 유지하기 위해 필수이며 현대 사회의 모든 온라인 활동을 안전하게 보호한다.
# 기밀성, 무결성, 인증, 부인방지
## 1. 기밀성
- 기밀성 (Confidentiality): 암호화된 데이터를 인가된 사용자만 볼 수 있도록 보호하여 비밀을 유지한다
  - Ex : 인터넷 뱅킹 시 계좌 비밀번호나 금융 데이터가 유출되어도 제3자가 내용을 알아볼 수 없도록 통신 구간을 암호화(HTTPS 등)하는 것
## 2. 무결성
- 무결성 (Integrity): 정보가 전송 과정에서 조작되거나 훼손되지 않았음을 보장다
  - Ex : 다운로드한 소프트웨어 설치 파일이나 패치가 중간에 해커에 의해 악성코드로 변조되지 않았음을 확인하기 위해 해시(Hash)값을 대조하는 것
## 3. 인증
- 인증 (Authentication): 디지털 서명 등을 통해 송수신자의 신원을 확인하고 신뢰성을 확보한다
  - Ex : 시스템에 접속할 때 아이디/패스워드 외에도 디지털 인증서나 생체 인식을 통해 현재 접근하려는 사람이 정당한 사용자 본인임을 증명하는 것
## 4. 부인방지
- 부인방지 (Non-repudiation): 메시지를 보낸 사람이 나중에 보낸 사실을 부인할 수 없게 한다
  - Ex : 전자 계약이나 온라인 송금 시 개인키로 전자 서명을 남겨 나중에 송신자가 나는 그런 문서에 서명하거나 돈을 보낸 적이 없다고 발뺌할 수 없게 만드는 것
# 평문, 암호화, 키
## 1. 평문
- 평문 (Plaintext): 암호화되기 전 이해할 수 있는 형태의 원본 데이터
## 2. 암호화
- 암호화 (Encryption): 알고리즘과 키를 이용해 평문을 암호문으로 변환하는 과정
## 3. 키
- 암호화, 복호화에 필요한 보안 매개체. 대칭키(동일한 키)와 공개키(비대칭키) 방식으로 나뉨
